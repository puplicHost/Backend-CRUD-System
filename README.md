# Backend CRUD System

A simple and functional **CRUD system** built with **Node.js**, **Express**, and **MongoDB** to manage users and courses.

## Features
- Create, Read, Update, and Delete (CRUD) operations for users and courses
- RESTful API design
- JSON request handling
- Environment variables with **dotenv**
- Clean and organized route structure
- Database connection using **Mongoose**

## Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
